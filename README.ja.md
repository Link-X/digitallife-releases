# digitallife（数字人生）

[English](README.md) · [简体中文](README.zh.md) · **日本語** · [한국어](README.ko.md) · [Français](README.fr.md) · [Deutsch](README.de.md) · [Español](README.es.md) · [Português](README.pt.md)

> ネイティブな macOS アクティビティモニター —— **すべてのデータはあなたの Mac の中だけに残ります。**

digitallife は、あなたの Mac の日々の動きを静かに記録します：どのアプリをどれくらい使ったか、ネットワークとバッテリー／電力の消費、ファイルの変更、インストール・アンインストールされたアプリなど…… そしてそれらを、洗練されたネイティブのメニューバーアプリで、わかりやすい日次／週次／月次レポートにまとめます。クラウドなし、アカウントなし、一切アップロードしません。

## ✨ 見られるもの

- **アプリ使用時間**と日別ランキング（アプリアイコン付き）
- **バッテリーと電力の推移**、アプリ別のエネルギー消費ランキング（オプション）
- アプリ別の**ネットワーク通信量**
- **ファイルアクティビティ**：作成／削除／変更 —— ディレクトリ、種類、発生元アプリ
- **CPU／メモリ**の上位リスト
- **ウインドウタイトルとアイドル時間**、メニューバー／バックグラウンド常駐アプリ
- アプリの**インストール／アンインストール**履歴
- **ディスク容量スキャン**（大きなフォルダ／ファイルの特定）と**不要ファイルのクリーンアップ**
- **日次／週次／月次レポート**、前期間とのスクリーンタイム比較
- データは日別／カテゴリ別に削除でき、**JSON／CSV エクスポート**に対応

## ⬇️ ダウンロードとインストール

1. **[Releases](https://github.com/Link-X/digitallife-releases/releases/latest)** から最新の `DigitalLife-<バージョン>.dmg` をダウンロード
2. DMG を開き、**「数字人生」を「アプリケーション」にドラッグ**
3. **ダブルクリックで起動** —— 本アプリは **Apple の公証（notarization）済み**のため、「壊れている」「開発元を確認できない」といったブロックは出ず、「右クリック→開く」も不要です
4. 初回起動後、メニューバーから「権限設定…」を開き、案内に従ってシステム権限を付与してください

### 必要な権限（必要に応じて付与。欠けても該当機能が制限されるだけで、他には影響しません）

| 権限 | 用途 |
|------|------|
| フルディスクアクセス | ブラウザの履歴など |
| アクセシビリティ | 最前面のアプリの識別 |
| 画面収録 | ウインドウタイトルの取得（**スクリーンショットは撮りません**） |
| オートメーション（システムイベント） | 最前面のアプリ／ウインドウ |

## 🔒 プライバシー

digitallife が最も大切にしている点です：

- **データの 100% がローカルに残ります**。あなたの Mac 上のローカルデータベースに書き込まれ、**ネットワークへの送信は一切ありません**
- **唯一の通信**は、あなたが**手動で「アップデートを確認」をクリックしたとき**だけ —— バージョン番号を取得するだけで、あなたのデータは一切送信しません。それ以外は完全にオフラインです
- より強力な保護が必要な場合は、設定で**データベース暗号化**を有効にできます（鍵はシステムのキーチェーンに保存されます）

## 🔄 アップデートの確認

アプリメニュー **「关于数字人生」（このアプリについて）→「检查更新」（アップデートを確認）**：手動で一度クリックすると、新しいバージョンがあればダウンロードリンクとともに通知されます。バックグラウンドでの定期確認や自動ダウンロードは行いません。
