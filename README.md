数字人生（digitallife）

▎ 一款原生的 macOS 电脑活动监控 App — 数据全部留在你自己的电脑里。

「数字人生」在后台安静地记录你这台 Mac 的日常：用了哪些应用、多久，网络与电量消耗、文件增删、装了卸了什么软件……然后用一个精致的原生菜单栏 App，把这些汇成清晰的日报 / 周报 / 月报。没有云、没有账号、不上传任何数据。

✨ 能看到什么

- 应用使用时长与每日排行（带图标）· 电量/功耗趋势与逐应用能耗（可选）· 网络流量按应用
- 文件活动 · CPU/内存 Top 榜 · 窗口/空闲、后台常驻应用 · 应用安装/卸载
- 磁盘空间扫描与垃圾清理 · 日/周/月报告 · 数据可清理、JSON/CSV 导出

💻 系统要求

macOS 13（Ventura）+ · Apple Silicon

⬇️ 下载与安装

下载最新 DMG → 拖进「应用程序」→ 双击启动（已 Apple 公证，无需右键打开）→ 菜单栏「权限设置…」授予权限。
权限表(完全磁盘/辅助功能/屏幕录制/自动化)+ 用途。

🔒 隐私

数据 100% 本地、无上报；唯一联网 = 你手动点检查更新；可选数据库加密。
