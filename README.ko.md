# digitallife（数字人生）

[English](README.md) · [简体中文](README.zh.md) · [日本語](README.ja.md) · **한국어** · [Français](README.fr.md) · [Deutsch](README.de.md) · [Español](README.es.md) · [Português](README.pt.md)

> 네이티브 macOS 활동 모니터 —— **모든 데이터는 당신의 Mac에만 남습니다.**

digitallife는 당신의 Mac에서 매일 일어나는 일을 조용히 기록합니다: 어떤 앱을 얼마나 오래 사용했는지, 네트워크와 배터리／전력 소비, 파일 변경, 설치되거나 제거된 앱 등…… 그리고 이 모든 것을 세련된 네이티브 메뉴 막대 앱으로 명확한 일간／주간／월간 리포트로 정리합니다. 클라우드 없음, 계정 없음, 아무것도 업로드하지 않습니다.

## ✨ 볼 수 있는 것

- **앱 사용 시간**과 일별 순위（앱 아이콘 포함）
- **배터리 및 전력 추이**, 앱별 에너지 소비 순위（선택 사항）
- 앱별 **네트워크 트래픽**
- **파일 활동**: 생성／삭제／수정 —— 디렉터리, 종류, 발생 앱
- **CPU／메모리** 상위 목록
- **창 제목과 유휴 시간**, 메뉴 막대／백그라운드 상주 앱
- 앱 **설치／제거** 기록
- **디스크 용량 스캔**（큰 폴더／파일 찾기）과 **불필요한 파일 정리**
- **일간／주간／월간 리포트**, 이전 기간과의 스크린 타임 비교
- 데이터는 날짜별／범주별로 삭제할 수 있으며 **JSON／CSV 내보내기** 지원

## ⬇️ 다운로드 및 설치

1. **[Releases](https://github.com/Link-X/digitallife-releases/releases/latest)**에서 최신 `DigitalLife-<버전>.dmg` 다운로드
2. DMG를 열고 **"数字人生"을 「응용 프로그램」으로 드래그**
3. **더블 클릭하여 실행** —— 이 앱은 **Apple 공증(notarization)을 받았으므로** "손상됨" "개발자를 확인할 수 없음" 같은 차단이 나타나지 않으며 "우클릭 → 열기"도 필요 없습니다
4. 처음 실행한 후 메뉴 막대에서 "권한 설정…"을 열고 안내에 따라 시스템 권한을 부여하세요

### 필요한 권한（필요에 따라 부여; 없어도 해당 기능만 제한되고 다른 기능에는 영향 없음）

## 🔒 개인정보 보호

digitallife가 가장 중요하게 여기는 점입니다:

- **데이터의 100%가 로컬에 남습니다**. 당신의 Mac에 있는 로컬 데이터베이스에 기록되며 **어떠한 네트워크 전송도 없습니다**
- **유일한 네트워크 요청**은 당신이 **직접 "업데이트 확인"을 클릭할 때**뿐입니다 —— 버전 번호만 가져오며 당신의 데이터는 전혀 전송하지 않습니다. 그 외에는 완전히 오프라인입니다
- 더 강력한 보호가 필요하면 설정에서 **데이터베이스 암호화**를 켤 수 있습니다（키는 시스템 키체인에 저장됨）

## 🔄 업데이트 확인

앱 메뉴 **"关于数字人生"（정보）→ "检查更新"（업데이트 확인）**: 수동으로 한 번 클릭하면 새 버전이 있을 경우 다운로드 링크와 함께 알려줍니다. 백그라운드 폴링이나 자동 다운로드는 하지 않습니다.
